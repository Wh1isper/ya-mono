# YA Claw Web

Vite + React + TypeScript web shell for YA Claw.

## Focus

- workspace browser
- profile selection
- session and run inspection
- backend handshake for the local runtime

## Development

```bash
cd apps/ya-claw-web
pnpm install
pnpm dev
```

Set `VITE_CLAW_BASE_URL` when the backend runs on a different origin.
